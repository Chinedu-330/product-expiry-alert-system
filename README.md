# Product Expiry Alert Management System

## Overview

The Product Expiry Alert Management System is a web-based application developed to help businesses monitor product expiry dates and receive timely notifications before products expire.

## Features

* User Authentication
* Product Registration
* Product Management
* Expiry Date Monitoring
* Email Notifications
* PDF Report Export
* Excel Report Export
* Database Backup

## Technologies Used

* Python
* Flask
* PostgreSQL
* HTML
* CSS
* JavaScript
* ReportLab
* Pandas
* OpenPyXL

## Installation

### Clone the repository

```bash
git clone https://github.com/Chinedu-330/product-expiry-alert-system.git
```

### Create a virtual environment

```bash
python -m venv venv
```

### Activate the virtual environment

```bash
venv\Scripts\activate
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
python app.py
```
